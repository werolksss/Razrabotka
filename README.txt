ПРОЕКТ: OnlineShopQt

Это полностью новый проект Qt Widgets для практической работы №1.
Тема: пользовательский интерфейс онлайн-магазина.

ЧТО ЕСТЬ В ПРОЕКТЕ:
1. Первая форма "Интернет магазин"
   - кнопка "Каталог товаров"

2. Форма "Каталог товаров"
   - список товаров
   - описание товара
   - стоимость
   - место под фото товара
   - кнопка "Заказать"

3. Форма "Заказ"
   - описание товара
   - стоимость
   - поля "Ваше имя" и "Телефон"
   - кнопка "Оформить заказ"
   - после оформления переход на первую форму

КАК ОТКРЫТЬ:
Лучше открыть файл OnlineShopQt.pro:
Qt Creator -> Файл -> Открыть файл или проект -> OnlineShopQt.pro

Если Qt Creator просит комплект, выбрать:
Qt 6.11.1 for macOS

Если открываешь через CMakeLists.txt, тоже должно работать, потому что CMakeLists.txt уже настроен под .ui файлы.
